# Startup-Success-prediction-ML
Machine Learning project to predict startup success using Python and Scikit-learn
This project builds a machine learning classification model to predict whether a startup will succeed or fail based on structured data. The model analyzes multiple factors such as funding, founder experience, skill distribution, and business attributes to make predictions.

---

## 🎯 Problem Statement
Startups have a high failure rate due to various uncertain factors like insufficient funding, lack of experience, and poor business decisions. This project aims to reduce that uncertainty by using machine learning to predict startup success based on historical and feature-based data.

---

## 📊 Dataset
- Total Features: 112  
- Data Type: Structured (Tabular)  
- Target Variable: Startup Success (Binary Classification)

### Key Features:
- Funding-related attributes  
- Founder experience  
- Skill distribution  
- Business model indicators  
- Company age and founding year  

---

## 🛠️ Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn   
- Matplotlib, Seaborn  

---

## ⚙️ Methodology

- Data cleaning and preprocessing  
- Handling missing values and categorical encoding  
- Feature scaling and selection  
- Exploratory Data Analysis (EDA)  
- Model training and comparison  
- Hyperparameter tuning  
- Model evaluation using multiple metrics  

---

## 🤖 Models Used

- Logistic Regression  
- Random Forest

### 🏆 Model Performance

- **Accuracy:** 95.8%  
- **F1 Score (Weighted Avg):** 0.96  
- The model shows strong performance with balanced precision and recall across both classes.
 
---

## 📈 Evaluation Metrics

- Accuracy  
- Precision  
- Recall  
- F1 Score  
- ROC-AUC  

---

## 🔍 Key Insights

- Funding is one of the most influential factors in startup success  
- Founder experience has a significant impact on outcomes  
- Skill distribution across the team strongly affects predictions  
- Data-driven approach improves decision-making in uncertain startup environments
